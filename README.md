# InoovaSaúde - Gestão de Clínicas

Sistema completo e moderno para gestão de clínicas médicas, agendamentos e automação de WhatsApp.

## Tecnologias Utilizadas

- **Frontend**: Vite + React + TypeScript
- **Estilização**: Tailwind CSS + shadcn-ui
- **Backend/Banco**: Supabase
- **Automação**: Supabase Edge Functions + WhatsApp API

## Como rodar localmente

1. Clone o repositório.
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Configure as variáveis de ambiente no arquivo `.env`.
4. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Deploy

O projeto está configurado para deploy via **Coolify** e as Edge Functions via **Supabase CLI**.
